### 색칠하기

재귀 형식에서 무조건 ㄱ, ㄴ 꼴로 나누어서
1, 2, 3, 4 분면을 생각해야 함

1. 색칠하기
   ```
   1, 2, 3, 4 분면중 한쪽 분면에 모든 매열이 0일떄
   ```
2. 색칠하기 못하는 경우
   ```
   1, 2, 3, 4 분면중 한쪽 분면에 하나라도 0이 아닌 놈이 있을때,
   ```

#### 최적화 하기

될것이다.
