### 케빈 베이컨의 6단계 법칙

#### 1). 플루이드 워셜 
플루이드 워셜 문제같다는 생각이 바로 떠올랐다
왜냐하면 모든 노드와 노드사이의 최단거리를 구해야 하므로.
`dp[i][j] = dp[i][k] + dp[k][j]`

그리고 주의할 점이
경유 인덱스 `k`는 가장 최상단 루프가 되야 한다.
```cpp
for(int k = 1; k <= N; k++) {
    for(int j = 1; j <= N; j++) {
        for(int i = 1; i <= N; i++) {
            ll dist = DP[i][k] + DP[k][j];
            if(DP[i][j] > dist) {
                DP[i][j] = dist;
                DP[j][i] = dist;
            }
        }
    }
}
```
