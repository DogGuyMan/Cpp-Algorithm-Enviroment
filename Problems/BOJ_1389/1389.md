### 케빈 베이컨의 6단계 법칙

플루이드 워셜이 바로 떠올랐는데 
`dp[i][j] = dp[i][k] + dp[k][j]`

그리고 주의할 점이
경유 인덱스 `k`는 가장 최상단 루프가 되야 한다.
```cpp
for(int k = 1; k <= N; k++) {
    for(int j = 1; j <= N; j++) {
        for(int i = 1; i <= N; i++) {
            ll dist = DP[i][k] + DP[k][j];
            if(DP[i][j] > dist) {
                DP[i][j] = dist;
                DP[j][i] = dist;
            }
        }
    }
}
```
